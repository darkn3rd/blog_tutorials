# Shell Library

This directory contains reusable shell functions and helper libraries.

Files in this directory are intended to be sourced by other scripts rather than executed directly.

Example:

```bash
source ../shell_lib/logging.sh
source ../shell_lib/aws.sh
```

Common uses include:

* Logging and output formatting
* Input validation
* AWS helper functions
* Kubernetes and EKS helper functions
* Reusable utility functions
